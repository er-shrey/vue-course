# Vue Course - To Do List

A simple to-do list app built with Vue 2 as a course exercise, using Bootstrap for styling. Components include a task list, an individual task, and a new-task form (`src/components/list.vue`, `task.vue`, `newtask.vue`).

## Tech stack

- Vue 2.6 (via `@vue/cli-service` 4.5)
- Bootstrap 4.5, jQuery 3.5, Popper.js (styling)
- ESLint (vue-cli-plugin-eslint)
- License: MIT

## Prerequisites

- Node.js 12+ (compatible with Vue CLI 4.5)
- npm

## Installation

```bash
npm install
```

## Running (development)

```bash
npm start
```

Compiles and hot-reloads for development (`vue-cli-service serve`).

## Build for production

```bash
npm run build
```

## Lint

```bash
npm run lint
```
